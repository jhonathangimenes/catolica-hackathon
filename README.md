# catolica-hackathon

## Introdução

```
A aplicação surge com o intuito de auxiliar alunos do Ensino Médio a escolher
a sua futura carreira profissional. Isso é feito através de sugestões de cursos
baseadas na vocação do aluno, essa vocação é definida através de suas notas, em
seu histórico escolar.
```

## Como funciona

```
O aluno terá suas médias em cada disciplina no Ensino Médio, o sistema calculará
a média geral desse aluno. Cada área de conhecimento (ciências exatas, humanas e biológicas)
terá um peso definido pela universidade para seus cursos. Essa média irá indicar se o aluno
pode se matricular no curso, bem como se será ofertado algum desconto nas mensalidades
para o mesmo.
```
